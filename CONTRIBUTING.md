# Contributing to MMU Simulator

Thank you for your interest in contributing to the MMU Simulator! Here are some guidelines:

## How to Contribute
1. **Fork** the repository on GitHub.
2. **Clone** your fork to your local machine.
3. **Commit** your changes to a new branch.
4. **Push** your branch and create a **Pull Request**.

## Technical Details
This project is a single-file application for ease of use in educational labs.
- **HTML/JS**: All logic is in `mmu_simulator_v2.html`.
- **CSS**: Tailwind CSS via CDN is used for UI.
- **Dependencies**: No build step is required. Just open the HTML file.

## Suggestions for Improvement
- Add support for Segmentation.
- Implement more Page Replacement algorithms (e.g., Clock, Second Chance).
- Enhance the Address Translation visualizer for multi-level paging.
- Improve accessibility (Aria labels, etc.).
